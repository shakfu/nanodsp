# Algorithm Guide

A hands-on guide to the signal processing algorithms in nanodsp, organized by category. Each section explains the technique and shows how to use it.

| Section | What it covers |
|---------|---------------|
| [Filters](filters.md) | Biquad, SVF, ladder, virtual analog, multi-order IIR |
| [Dynamics](dynamics.md) | Compression, limiting, gating, AGC, parallel/multiband |
| [Saturation](saturation.md) | Soft/hard/tape clipping, antialiased waveshaping |
| [Reverb](reverb.md) | FDN, Schroeder, Moorer, STK, ReverbSc |
| [Modulation](modulation.md) | Chorus, flanger, phaser, tremolo, autowah |
| [Spectral](spectral.md) | STFT, time stretch, pitch shift, denoise, EQ match |
| [Synthesis](synthesis.md) | Oscillators, FM, noise, drums, physical modeling, STK |
| [Analysis](analysis.md) | Loudness, spectral features, pitch, onsets, resampling |
| [Composed Effects](composed.md) | Mastering chains, creative effects, delay networks |
| [Granular](granular.md) | Granular synthesis via GrainflowLib |
| [Streaming](streaming.md) | Ring buffers, block processors, overlap-add |
